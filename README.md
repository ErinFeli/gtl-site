# GTL Construções e Serviços - Site Institucional

Site institucional moderno e responsivo para a GTL Construções e Serviços, empresa líder em soluções de engenharia.

## 🚀 Características

- **Design Moderno**: Interface clean e profissional com foco em UX
- **Totalmente Responsivo**: Otimizado para desktop, tablet e mobile
- **Performance Otimizada**: Carregamento rápido e animações suaves
- **SEO Friendly**: Estrutura semântica e meta tags otimizadas
- **Acessível**: Seguindo as melhores práticas de acessibilidade web

## 📋 Seções do Site

1. **Hero Section**: Apresentação impactante da empresa
2. **Quem Somos**: Informações sobre a empresa e seus valores
3. **Serviços**: Showcase dos principais serviços oferecidos
4. **Números**: Estatísticas e alcance da empresa
5. **CTA**: Call-to-action para conversão
6. **Contato**: Formulário e informações de contato
7. **Footer**: Links rápidos e redes sociais

## 🚀 Como Usar

### Instalação Local

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Ou use um servidor local como Live Server (VSCode)

### Hospedagem

Este site é 100% estático e pode ser hospedado em qualquer serviço:

- **GitHub Pages**: Gratuito e simples
- **Netlify**: Deploy automático
- **Vercel**: Excelente performance
- **Hostgator/cPanel**: Hospedagem tradicional

## ⚙️ Funcionalidades JavaScript

- **Menu Mobile**: Navegação responsiva com hambúrguer menu
- **Scroll Suave**: Navegação suave entre seções
- **Active Link**: Link ativo baseado na seção visível
- **Contadores Animados**: Números da empresa aparecem com animação
- **Lazy Loading**: Carregamento otimizado de imagens
- **Animações de Entrada**: Elementos aparecem ao scroll
- **Formulário Integrado**: Envio direto para WhatsApp

## 📱 Responsividade

O site é totalmente responsivo com breakpoints em:

- **Desktop**: > 968px
- **Tablet**: 768px - 968px
- **Mobile**: < 768px
- **Small Mobile**: < 480px

## 🔧 Customização

### Cores

Edite as variáveis CSS no topo de `styles.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #f59e0b;
    /* ... */
}
```

### Conteúdo

Todo o conteúdo está em `index.html` e pode ser facilmente editado:

- Textos nas seções
- Links de contato
- Imagens (URLs do Unsplash)
- Redes sociais

### Imagens

Substitua os URLs das imagens por:
- Imagens locais (salve na pasta `images/`)
- URLs do seu servidor
- CDN personalizada
